## 1. ログイン方法

### サイトURL
```
https://sample-manager2026.web.app
```

### ログイン方法は2種類

#### A. Googleアカウントの場合
1. 「**Googleでログイン**」ボタンをクリック
2. Googleアカウントを選択
3. ログイン完了

#### B. その他のメールアドレスの場合

**初回（新規登録）:**
1. 「**新規登録**」リンクをクリック
2. メールアドレスを入力
3. パスワードを入力（6文字以上）
4. パスワード（確認）を再入力
5. 「**新規登録**」ボタンをクリック

**2回目以降:**
1. メールアドレスを入力
2. パスワードを入力
3. 「**ログイン**」ボタンをクリック

> **注意**: 管理者に事前にメールアドレスを登録してもらう必要があります。
> 登録されていない場合は「アクセスが許可されていません」と表示されます。

## 2. サンプル一覧の使い方

### 画面の見方

| 項目 | 説明 |
|------|------|
| **No** | サンプル番号（クリックで詳細表示） |
| **臨床診断** | 診断名 |
| **年齢** | 患者年齢 |
| **性別** | 患者性別 |
| **検体** | 血清 または 髄液 |
| **採取日** | 検体採取日 |
| **初期保存量** | 保存用の初期量（クリックで編集可能） |
| **初期解析用** | 解析用の初期量（クリックで編集可能） |
| **残量** | 保存/解析用の現在残量 |

### 残量の色の意味

| 色 | 意味 |
|----|------|
| **緑** | 十分にあり |
| **オレンジ** | 残量少（設定した閾値以下） |
| **赤背景「なし」** | 残量なし（0） |

### 検索・フィルタ

#### キーワード検索
- 検索ボックスにNo、診断名、識別番号などを入力
- 入力するとリアルタイムで絞り込み

#### 検体で絞り込み
- 「すべての検体」「血清のみ」「髄液のみ」から選択

#### 残量で絞り込み
- 「すべての残量」「残量少」「残量なし」から選択

#### 番号範囲で絞り込み
1. 「開始」に最初の番号を入力（例: `190001`）
2. 「終了」に最後の番号を入力（例: `190100`）
3. 自動的にその範囲のサンプルのみ表示
4. 「クリア」で範囲指定を解除

### 統計カード
画面上部のカードをクリックすると、該当するサンプルが絞り込まれます：
- **総サンプル数**: 全件表示
- **血清サンプル**: 血清のみ表示
- **髄液サンプル**: 髄液のみ表示
- **残量少**: 残量が少ないサンプルのみ表示
- **残量なし**: 残量がないサンプルのみ表示

## 3. 使用を記録する

### 手順

1. サンプル一覧で該当サンプルの「**使用**」ボタンをクリック
2. 以下を入力：

| 項目 | 説明 |
|------|------|
| **検体・区分** | 保存血清/解析用血清/保存髄液/解析用髄液 から選択 |
| **使用量** | 使用したμl数を入力 |
| **用途** | 例: BioPlex測定、ELISA など |
| **日付** | 使用した日付（デフォルトは今日） |

3. 「**記録する**」ボタンをクリック
4. 残量が自動で再計算されます

> **注意**: 残量を超える使用量は入力できません。

---

## 4. 初期量を変更する

### 方法1: 一覧から直接編集（推奨）

1. サンプル一覧の「**初期保存量**」または「**初期解析用**」のセルをクリック
2. 数値を入力
3. **Enter**キー または セル外をクリックで保存
4. **Esc**キー でキャンセル

### 方法2: 詳細画面から編集

1. サンプルのNoをクリックして詳細画面を開く
2. 「**使用を記録**」の横にある操作から変更

## 5. データをエクスポートする

### 検索結果をExcelで出力

1. 検索やフィルタでサンプルを絞り込む
2. 「**検索結果をExcel出力**」ボタン（緑）をクリック
3. `検索結果_○件_日付.xlsx` がダウンロードされます

**出力される項目:**
- No、臨床診断、年齢、性別、識別番号
- 検体、採取日
- 初期保存量、初期解析用
- 保存残量、解析用残量

### 全データバックアップ（管理者向け）

1. 「**設定**」タブを開く
2. 「データ管理」セクションで：
   - 「**Excelでバックアップ**」: サンプル一覧＋使用履歴
   - 「**JSONでバックアップ**」: 全データ（復元用）

## 6. データを取り込む（管理者向け）

### Excelファイルからインポート

1. 「**データ取込**」タブを開く
2. Excelファイル（.xlsx）をドラッグ＆ドロップ、または「ファイルを選択」
3. プレビューで内容を確認
4. 「**インポート実行**」をクリック

**対応フォーマット:**
- 都医学研 sample list のExcelファイル
- 「Sample」シートからデータを読み込み

**読み込まれる項目:**
- No、臨床診断、年齢、性別、識別番号、検体、採取日

> **注意**: 既に同じNoと検体の組み合わせが存在する場合はスキップされます。

## 7. 設定（管理者向け）

### 残量アラート設定

残量が少ないと判定する閾値を設定します。

1. 「**設定**」タブを開く
2. 「残量アラート設定」セクションで：
   - **保存用**: 保存血清/髄液の閾値（μl）
   - **解析用**: 解析用血清/髄液の閾値（μl）
3. 「**閾値を保存**」をクリック

設定は全ユーザーに共有されます。

### 許可ユーザー管理

アクセスを許可するユーザーを管理します。

#### ユーザーを追加
1. 「**設定**」タブを開く
2. 「許可ユーザー管理」セクション
3. メールアドレスを入力
4. 「**追加**」をクリック

#### ユーザーを削除
1. ユーザー一覧で該当ユーザーの「**削除**」をクリック

> **注意**: 自分自身は削除できません。

## 使用履歴の確認

1. 「**使用履歴**」タブを開く
2. 日付範囲やサンプルNoで絞り込み可能
3. 誰がいつ、どれだけ使用したかを確認できます

| 項目 | 説明 |
|------|------|
| **日時** | 使用日 |
| **サンプルNo** | クリックで詳細表示 |
| **検体種類** | 血清/髄液 |
| **区分** | 保存/解析用 |
| **使用量** | 使用したμl数 |
| **用途** | 記録した用途 |
| **使用者** | 記録したユーザー |
| **残量** | 使用後の残量 |

## 画像の紐付け

紙のアンケート用紙の写真などをサンプルに紐付けできます。

### 画像を追加
1. サンプルのNoをクリックして詳細画面を開く
2. 「**画像を追加**」ボタンをクリック
3. 画像ファイルを選択
4. 「**アップロード**」をクリック

### 画像を確認
- サンプル詳細画面にサムネイルが表示されます
- クリックで拡大表示

## よくある質問

### Q. ログインできません
**A.** 管理者にメールアドレスの登録を依頼してください。

### Q. 使用を記録できません
**A.** 残量を超える使用量を入力していないか確認してください。

### Q. 画面が更新されません
**A.** ブラウザをリロード（Cmd+R または Ctrl+R）してください。

### Q. 初期量を間違えて入力しました
**A.** 一覧の初期量セルをクリックして修正できます。

### Q. 使用履歴を削除したい
**A.** 現在、履歴の削除機能はありません。管理者にお問い合わせください。
